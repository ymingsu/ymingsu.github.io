Mathematica退出窗口不完整的解决方法

发表于 2020-10-22 分类于 工具 阅读次数：

在退出 Mathematica 时，有可能出现退出窗口不完整的情况，而我在 Win10 电脑上安装过几次 Mathematica 11 或 12 ，则是每次都出现该问题，如图所示：

该问题的解决方法为：以管理员身份启动 Mathematica，然后在新建的笔记本中运行下面的代码即可：

(*==需要以管理员身份运行==*)
(*退出对话框路径*)
path = $InstallationDirectory <>
   "/SystemFiles/FrontEnd/TextResources/MathematicaClosingDialog.nb";
(*备份*)
If[! FileExistsQ[path <> ".bk"], CopyFile[path, path <> ".bk"]];
(*导入*)
nb = NotebookPut[Import[path]];
(*修改窗口尺寸*)
SetOptions[
  nb, {WindowSize -> {600, All}, WindowElements -> {"MenuBar"}}];
(*保存*)
Export[path, nb];
